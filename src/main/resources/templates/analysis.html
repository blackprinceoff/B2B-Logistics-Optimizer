Математичний аналіз ефективності евристики

Оцінка збіжності, чутливості та стохастичне моделювання методом Монте-Карло

1. Оцінка ефективності (Optimality Gap)

Порівняння розробленого евристичного алгоритму зі стандартним "жадібним" підходом (Look-Ahead = 0).

\[ Gap = \frac{Z_{heuristic} - Z_{baseline}}{|Z_{baseline}|} \times 100\% \]

Базовий прибуток (Baseline): 0 грн

Оптимізований прибуток: 0 грн

Відносне покращення (Gap): 0%

2. Аналіз чутливості горизонту планування

Залежність значення цільової функції від параметра LOOK_AHEAD.

3. Стохастичне моделювання (Монте-Карло)

Розподіл прибутку при наявності випадкових подій (ймовірність поломки \(P_{br} = 0.01\), відмова клієнта \(P_{ns} = 0.03\)).

Математичне сподівання: \( M[X] \) = 0
Дисперсія: \( D[X] \) = 0
Сер. квадр. відхилення: \( \sigma = \sqrt{D[X]} \) = 0
